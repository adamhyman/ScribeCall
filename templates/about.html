About ScribeCall

What is ScribeCall?

ScribeCall is a voice to text program written in Python.

Users can connect into voice chat room and talk with one another.

All voice inputs are converted into text for the other person.

ScribeCall converts the voice to text in real time.

This program will help users who have trouble hearing.

How to use ScribeCall

To use this program, just connect to a voice chat room and start talking to someone else.

One person will talk and the speech will get converted into text for the other person to read.

This will allow both users to easily understand what is being said.

When you are done with the call, simply press the disconnect button.
